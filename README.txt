========================================
AUDIO ANALYZER GOTHIC - INSTALLATION PWA
========================================

ÉTAPE 1 - HÉBERGER SUR GITHUB PAGES (gratuit, 10 min)
------------------------------------------------------
1. Crée un compte sur github.com (si t'en as pas)
2. Nouveau repository → nomme-le "audio-analyzer" → Public → Create
3. "uploading an existing file" → glisse TOUS les fichiers de ce zip:
   - index.html
   - manifest.json
   - sw.js
   - icon-192.png
   - icon-512.png
4. Commit changes
5. Settings → Pages → Source: "Deploy from a branch" → Branch: main → Save
6. Attends 2 min. Ton app est à:
   https://TON-USERNAME.github.io/audio-analyzer/

ÉTAPE 2 - INSTALLER SUR TON IPHONE
------------------------------------------------------
1. Ouvre l'URL dans SAFARI (pas Chrome!)
2. Bouton Partager (carré avec flèche)
3. "Sur l'écran d'accueil"
4. Ajouter

C'est tout. Icône gothique sur ton écran d'accueil,
plein écran, fonctionne offline après la première ouverture.

MISES À JOUR
------------------------------------------------------
Quand je te donne une nouvelle version de index.html:
remplace le fichier sur GitHub, l'app se met à jour toute seule
(peut prendre 1-2 ouvertures à cause du cache).
