# Audio Analyzer

Audio Analyzer is a music analysis and mix review app designed to help producers compare a mix against a reference track, detect technical issues, and get AI-powered suggestions for better decisions.

## What it does

This app helps you analyze the musical balance and technical quality of a track by checking:

- Full music spectrum in 14 bands.
- Stereo/mono compatibility with phase correlation and width by band.
- Crest factor and dynamics using peak vs RMS analysis.
- A/B comparison between your own bounce versions.
- Resonance detection for narrow problematic peaks.
- Section-aware analysis, such as verse vs chorus.
- Mix progress history with JSON export.
- AI suggestions for corrective actions and mix improvements.
- Reference vs mix comparison to help you match industry standards.

## Main features

- 14-band full-spectrum visual analyzer.
- Stereo width and phase correlation monitoring.
- Dynamic range and crest factor analysis.
- Reference track comparison.
- Mix-to-mix bounce comparison.
- Resonance spotting with suggested EQ corrections.
- Section-based analysis for deeper mix decisions.
- Exportable analysis history.
- AI-assisted feedback and recommendations.

## Why this project exists

Most free tools only show part of the picture. Audio Analyzer is built to give a more complete view of a mix by combining spectrum analysis, stereo checks, dynamics, references, and AI guidance in one place.

## Getting started

1. Open the project in your browser or development environment.
2. Load a mix file and a reference track.
3. Review the spectrum, stereo, dynamics, and resonance data.
4. Use the AI suggestions to decide what to fix first.
5. Save or export your analysis for future comparison.
